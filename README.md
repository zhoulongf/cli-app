# 基于 yeoman-generator 脚手架的城管前端项目初始化工具

## 目前支持平台项目

- pc 端项目脚手架
- 移动端 h5 项目脚手架
- 小程序项目脚手架
- 基于 rollup 的 lib 库脚手架

## 运行步骤

- 1、全局安装`@zlf/cli-app`
- 2、运行 `cli-app` 输入根据提示初始化项目
